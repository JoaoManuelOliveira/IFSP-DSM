## 🎲 Projeto: Jogo de Dados Mobile

Este projeto foi desenvolvido como parte da disciplina de **Desenvolvimento de Dispositivos Móveis**, no **4º ano do curso do IFSP**. A proposta da atividade foi aplicar, na prática, conceitos fundamentais do desenvolvimento Android utilizando **Kotlin**, explorando tanto a estrutura quanto os componentes visuais de uma aplicação mobile.

O aplicativo consiste em um **jogo de dados**, onde o usuário pode interagir com a interface para realizar lançamentos e visualizar os resultados de forma dinâmica. Durante o desenvolvimento, foram utilizados elementos essenciais da construção de layouts, como o **LinearLayout**, além de outros componentes de interface que contribuem para a organização e responsividade da aplicação.

### 🛠️ Tecnologias e conceitos utilizados

* Kotlin
* Android Studio
* LinearLayout e organização de layouts
* Manipulação de eventos (cliques)
* Atualização dinâmica da interface
* Estrutura básica de aplicações mobile

### 🎯 Objetivos do projeto

* Praticar o desenvolvimento de interfaces em Android
* Compreender a interação entre lógica e UI
* Aplicar conceitos básicos de programação em Kotlin
* Desenvolver uma aplicação funcional simples

### 📱 Funcionalidades

* Simulação de lançamento de dados
* Exibição de resultados em tempo real
* Interface simples e intuitiva
